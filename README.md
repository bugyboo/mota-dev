# mota-dev
For building Mota Development Env

# Download the zip file
- Using CURL
```
curl -L -O https://github.com/bugyboo/mota-dev/archive/refs/heads/main.zip
```

- Unzip the file
```
tar -xf main.zip
```
- Run the script
```
cd .\mota-dev-main\
// For Windows
./install.bat
// OR
install
// For shell
./install_osx.sh
