# mota-dev
For building Mota Development Env

# Download the zip file
- Using CURL
```
curl -L -O https://github.com/bugyboo/mota-dev/archive/refs/heads/main.zip
```

- Unzip the file
```
tar -xf main.zip
```
## Run the script on Windows
```
cd .\mota-dev-main\
./install.bat
```
## Run the script on Linux / MacOS
```
cd ./mota-dev-main
./install_osx.sh
